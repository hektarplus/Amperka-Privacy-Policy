Amperka is a simple Android application that provides real-time information about your smartphone’s charging process (battery voltage, current, temperature, and level).

The app does not collect, store, or share any personal data. All measurements are performed locally on your device and are never transmitted to any external server.

Data usage

No user accounts are required.

No personal or location data is collected.

The app works entirely offline, except when displaying ads (if applicable).
